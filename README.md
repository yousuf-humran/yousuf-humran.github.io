## âš™ï¸ Customization

To personalize this portfolio for yourself:

- **Information**: Update the ```<h1>``` ```<p>``` and ```<aside>``` tags in index.html with your name and bio.

- **Projects**: Edit the ```.project-list``` section with your own GitHub links and project thumbnails.

- **Experience**: Update the ```.exp-list``` with your professional history.

- **Forms**: Replace the Formspree ID ```mpqqplyp```  in the ```<form>```  action attribute with your own to receive emails.
